#Chrome 平台

Google Chrome是一个由Google开发的网页浏览器。在 Chrome 平台上，你可以做丰富的 HTML5 应用，也可以基于 Chrome 开发浏览器插件、主题等，同时随着 ChromeBook 在教育、办公行业的发展，也为开发者提供了大量的商业变现机会，同时随着 App Runtime for Chrome  的发布，侬也可以在 Chrome 上运行 Android 程序了。
