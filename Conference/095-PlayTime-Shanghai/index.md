# PlayTime@Shanghai

这个活动为中国开发者们带来了如何在Google Play上进行应用发布所应该关注的东西，包括最新设计指南Material Design、最新开发工具和技术的分享、以及最重要的、也是中国开发者们目前还不是普遍理解的，在Play上进行应用发布所必须要遵循的各种规则和指南。

这些视频是对利用Google Play生态系统从设计到开发到发布到赢利的一个全面的解释，具有大量的最新技术发展信息以及如何帮你的应用和游戏走向国际市场的实际指南。希望所有已经在或者正准备在Play进行应用发布的开发者们，把这些视频仔细看一遍、特别是Dana Yang的演讲。[via](http://www.chinagdg.com/thread-6742-1-1.html)
